# IRVe_bot
Телеграм-бот "Неправильные глаголы? Легко! (IRVe)". Создан для помощи в изучении неправильных глаголов английского языка!

# Возможности
Поиск неправильного глагола по любой форме
Вывод списка всех неправильных глаголов
# О боте
Создавался с октября 2025 по февраль 2026. В настоящее время в активной разработке. Бот постоянно улучшается, появляются новые функции!
